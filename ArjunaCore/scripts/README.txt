Description of Scripts
----------------------

*-setup-env.sh  - BASH shell script which sets up environment for ArjunaJTS product
*-setup-env.bat - DOS batch file which sets up environment for ArjunaJTS product

start-recovery-manager.bat 		- DOS batch file which starts the recovery manager
start-transaction-service.bat	- DOS batch file which starts the transaction service
