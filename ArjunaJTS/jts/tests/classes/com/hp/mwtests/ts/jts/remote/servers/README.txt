For  OrbixWeb, run putserver.orbix  for each server with the following
parameters, in the following order, with the exception of HammerServer:

i) the server name (e.g., Grid)
ii) the server's class name (e.g., GridServer)

See the  client   directories for the  servers   they require  and the
parameters.

For HammerServer and  OrbixWeb  run puthammer.orbix  and then run  the
server twice with 1 and 2 as the parameters.

For other ORBs simply run runserver with the name of the server.
